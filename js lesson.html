شرح دورة أساسيات js

JavaScript (JS) هي لغة برمجة قوية تُستخدم لتطوير تطبيقات الويب، سواء على جانب العميل (Frontend) أو الخادم (Backend). إذا كنت مبتدئًا، هنا الأساسيات التي يجب أن تعرفها:

لدرس الأول: أساسياتjs

المتغيرات (Variables) المتغيرات تُستخدم لتخزين القيم. في JavaScript، يمكنك تعريف المتغيرات باستخدام var (قديمة)، أو let (مُفضّلة)، أو const (للثوابت).

شرح مفصل:
  • ما هي js؟هي لغة برمجة قوية تُستخدم لتطوير تطبيقات الويب، .
  • لماذا نحتاج js؟ لأنها تعتبر الركيزة الأساسية في تطوير تطبيقات الويب الحديثة
  • كيفية استخدام js: يمكن استخدام JavaScript بسهولة لأنها تُدمج مع صفحات الويب
  • أمثلة على خصائص js:
    • طباعة رسالة في وحدة التحكم console.log("مرحبًا بك في JavaScript!");
    • التعامل مع المتغيراتlet name = "علي"; let age = 25; let isStudent = true; console.log(`الاسم: ${name}, العمر: ${age}, طالب: ${isStudent}`);
    • استخدام الشروطlet score = 85; if (score >= 90) { console.log("ممتاز!"); } else if (score >= 75) { console.log("جيد جدًا"); } else { console.log("حاول مرة أخرى!"); }

      الدرس الثالث: التصفح في js

      التصفح في JavaScript (Navigation) يشير إلى كيفية التنقل والتفاعل مع الصفحة الحالية أو الصفحات الأخرى في المتصفح باستخدام الكود. JavaScript توفر واجهة قوية تُعرف بـ واجهة كائن المتصفح (Browser Object Model - BOM)، والتي تتيح التحكم الكامل في التصفح.

      شرح مفصل:
      • 1. الانتقال إلى صفحة أخرىwindow.location.href = "https://www.example.com";
      • إعادة توجيه الصفحة (Redirect):window.location.replace("https://www.example.com"); javascript نسخ الكود
      • إعادة تحميل الصفحة: javascript نسخ الكودwindow.location.reload(); // يعيد تحميل الصفحة الحالية
      مثال:
      window.history.forward(); window.history.go(-2); // الرجوع صفحتين للخلف window.history.go(2); // التقدم صفحتين للأمام

      الدرس الرابع: الترابط في js

      الترابط في JavaScript يشير إلى كيفية ربط الكائنات والدوال والقيم ببعضها البعض، وكيف يمكن التحكم في نطاق التنفيذ وسياق الدوال (context). الترابط يعتبر من المفاهيم الأساسية لفهم كيفية عمل JavaScript بشكل عميق.

      شرح مفصل:
      • 1. الترابط الضمني (Implicit Binding)
      • 2. الترابط الصريح (Explicit Binding)
      • 3. الترابط الافتراضي (Default Binding)
        • يحدث عندما يتم استدعاء الدالة عبر كائن (object). الكائن الذي يسبق النقطة (.) هو الذي يُعتبر سياق التنفيذ (this).
        • يتم تحديد قيمة this بشكل صريح باستخدام الدوال: call، apply، و bind.
        • إذا لم يكن هناك سياق (كائن) محدد، فإن this تشير إلى الكائن global (في المتصفح: window).
      مثال:
      const user = { name: "علي", greet: function () { console.log(`مرحبًا، ${this.name}`); } }; user.greet(); // مرحبًا، علي
      const user = { name: "سارة" }; function greet() { console.log(`مرحبًا، ${this.name}`); } greet.call(user); // مرحبًا، سارة
      function greet() { console.log(this); } greet(); // في المتصفح: Window، في Node.js: global
