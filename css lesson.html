شرح دورة أساسيات CSS

في هذه الدورة، سنتعلم أساسيات لغة HTML، التي تعتبر الأساس لبناء صفحات الويب. سنغطي على المفاهيم الأساسية مثل بنية الوثيقة، العناصر، الصيغ، وكيفية استخدامها لإنشاء صفحات ويب تفاعلية وجذابة. سنبدأ بالمفاهيم النظرية وننتقل إلى أمثلة عملية لتسهيل التطبيق.

الدرس الأول: أساسيات CSS

CSS (Cascading Style Sheets) هي لغة تنسيق تُستخدم لتحديد مظهر عناصر HTML. تُستخدم CSS لتحديد الألوان، الأحجام، الخطوط، وغيرها من الخصائص البصرية.

شرح مفصل:
  • ما هي CSS؟ CSS هي لغة تنسيق تُستخدم لتحديد مظهر عناصر HTML.
  • لماذا نحتاج CSS؟ CSS تُستخدم لتحسين مظهر الصفحات الويبية وتحديد مظهر العناصر بشكل موحد.
  • كيفية استخدام CSS: يمكن استخدام CSS بواسطة عنصر <style> داخل وثيقة HTML أو بواسطة ملف CSS خارجي.
  • أمثلة على خصائص CSS:
    • color: لتحديد لون النص.
    • font-size: لتحديد حجم الخط.
    • background-color: لتحديد لون الخلفية.
    • margin: لتحديد المسافة خارج العنصر.
    • padding: لتحديد المسافة داخل العنصر.
مثال:
<style>
body {
font-family: Arial, sans-serif;
background-color: #f0f0f0;
color: #333;
margin: 0;
padding: 20px;
}
h1 {
color: #ff0000;
}
</style>

الدرس الثاني: المحددات في CSS

المحددات في CSS تُستخدم لتحديد قيم محددة للخصائص. يمكن أن تكون المحددات قيمًا مطلقة مثل px (بكسل) أو % (نسبة) أو em (وحدة الخط).

شرح مفصل:
  • المحددات المطلقة:
    • px (بكسل): وحدة قياس مطلقة تُستخدم لتحديد الأحجام والمسافات.
    • % (نسبة): نسبة من الحجم الكلي للعنصر الأب.
    • em (وحدة الخط): وحدة قياس نسبية تعتمد على حجم الخط الحالي.
  • أمثلة على استخدام المحددات:
    • width: 100px; تحديد عرض العنصر بـ 100 بكسل.
    • margin: 20%; تحديد المسافة الخارجية للعنصر بنسبة 20% من الحجم الكلي للعنصر الأب.
    • padding: 1em; تحديد المسافة الداخلية للعنصر بوحدة الخط الحالية.
مثال:
<style>
.container {
width: 80%;
margin: 20px;
padding: 1em;
}
</style>

الدرس الثالث: التصفح في CSS

التصفح في CSS تُستخدم لتحديد كيفية عرض العناصر على الشاشة. يمكن أن تكون التصفح إما عادية أو ناعمة أو مطبوعة.

شرح مفصل:
  • التصفح العادية: تُستخدم لعرض العناصر بشكل عادي على الشاشة.
  • التصفح الناعمة: تُستخدم لعرض العناصر بشكل ناعم على الشاشة.
  • التصفح المطبوعة: تُستخدم لعرض العناصر بشكل مطبوع على الشاشة.
  • أمثلة على التصفح:
    • display: block; تحديد التصفح العادية.
    • display: inline; تحديد التصفح الناعمة.
    • display: none; تحديد التصفح المطبوعة.
مثال:
<style>
.hidden {
display: none;
}
.inline {
display: inline;
}
</style>

الدرس الرابع: الترابط في CSS

الترابط في CSS تُستخدم لتحديد كيفية تفاعل العناصر مع الفأرة. يمكن أن تكون الترابط إما داخلية أو خارجية.

شرح مفصل:
  • الترابط الداخلية: تُستخدم لتحديد كيفية تفاعل العناصر مع بعضها البعض.
  • الترابط الخارجية: تُستخدم لتحديد كيفية تفاعل العناصر مع العناصر الأخرى.
  • أمثلة على الترابط:
    • margin-top: 20px; تحديد المسافة العلوية للعنصر بـ 20 بكسل.
    • padding-left: 10px; تحديد المسافة اليسرى الداخلية للعنصر بـ 10 بكسل.
    • border: 1px solid #000; تحديد حد حول العنصر بسمك 1 بكسل ولون أسود.
مثال:
<style>
.box {
margin-top: 20px;
padding-left: 10px;
border: 1px solid #000;
}
</style>

الدرس الخامس: الأنماط المتقدمة

CSS توفر مجموعة من الأنماط المتقدمة التي تتيح لك تحديد عناصر بطرق دقيقة ومرنة. هذه الأنماط تشمل الأنماط النسبية والأنماط المستندة إلى السمات.

شرح مفصل:
  • محددات النسب (Child/Descendant Selectors): تُستخدم لتحديد العناصر النسبية داخل عنصر آخر.
  • محددات السمات (Attribute Selectors): تُستخدم لتحديد العناصر بناءً على السمات الخاصة بها.
  • محددات الأنماط الافتراضية: مثل `:hover`, `:focus` التي تُستخدم لتحديد التفاعلات مع العناصر.
مثال:
<style>
/* محدد النسب */
ul li {
color: blue;
}
/* محدد السمات */
input[type="text"] {
border: 1px solid green;
}
/* محدد عند التفاعل */
button:hover {
background-color: yellow;
}
</style>

الدرس السادس: العمل مع الخطوط

التحكم في الخطوط هو جزء مهم في تصميم الويب. باستخدام CSS، يمكن تغيير نوع الخط، حجمه، وزخرفته.

شرح مفصل:
  • font-family: لتحديد نوع الخط.
  • font-size: لتحديد حجم الخط.
  • font-weight: لتحديد وزن الخط (خفيف، عادي، عريض).
  • text-decoration: لتحديد زخارف النص مثل الخط السفلي أو العلوي.
مثال:
<style>
h1 {
font-family: 'Arial', sans-serif;
font-size: 24px;
font-weight: bold;
text-decoration: underline;
}
p {
font-family: 'Times New Roman', serif;
font-size: 18px;
font-weight: normal;
}
</style>

الدرس السابع: الألوان والخلفيات

CSS تتيح التحكم في الألوان والخلفيات لجعل التصميمات أكثر جاذبية وتفاعلية.

شرح مفصل:
  • color: لتحديد لون النص.
  • background-color: لتحديد لون خلفية العنصر.
  • background-image: لإضافة صورة كخلفية.
  • background-repeat: للتحكم بتكرار الصورة الخلفية.
  • background-position: لتحديد موضع الصورة الخلفية.
مثال:
<style>
body {
background-color: #e0f7fa;
color: #004d40;
}
.box {
background-image: url('background.jpg');
background-repeat: no-repeat;
background-position: center;
}
</style>

الدرس الثامن: التخطيطات باستخدام Flexbox

Flexbox نظام قوي في CSS يُستخدم لإنشاء تخطيطات مرنة ومتكيفة مع مختلف أحجام الشاشات.

شرح مفصل:
  • display: flex; تفعيل Flexbox على العنصر الأب.
  • justify-content: محاذاة العناصر أفقيًا (start, center, space-between).
  • align-items: محاذاة العناصر عموديًا.
  • flex-direction: تحديد اتجاه العناصر (row, column).
مثال:
<style>
.container {
display: flex;
justify-content: center;
align-items: center;
height: 100vh;
background-color: #f0f0f0;
}
.item {
margin: 10px;
padding: 20px;
background-color: #3e0292;
color: white;
}
</style>

الدرس التاسع: التخطيطات باستخدام Grid

CSS Grid هو نظام تصميم تخطيطي ثنائي الأبعاد لإنشاء تخطيطات معقدة بسهولة.

شرح مفصل:
  • display: grid; تفعيل Grid على العنصر الأب.
  • grid-template-columns: تحديد أعمدة الشبكة.
  • grid-template-rows: تحديد صفوف الشبكة.
  • gap: تحديد المسافة بين الأعمدة والصفوف.
مثال:
<style>
.grid-container {
display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 20px;
}
.grid-item {
background-color: #6200ea;
color: white;
padding: 20px;
text-align: center;
}
</style>

الدرس العاشر: الظلال

CSS تتيح إضافة الظلال إلى النصوص والعناصر لجعل التصميم أكثر تميزًا.

شرح مفصل:
  • text-shadow: لإضافة ظل إلى النصوص.
  • box-shadow: لإضافة ظل إلى المربعات والعناصر.
مثال:
<style>
h1 {
text-shadow: 2px 2px 5px #888;
}
.box {
box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.5);
}
</style>

الدرس الحادي عشر: الحركات والتحولات

الحركات والتحولات تُستخدم لجعل العناصر تتغير بطريقة سلسة.

شرح مفصل:
  • transition: لإضافة تأثيرات سلسة عند تغيير الخصائص.
  • transform: لتدوير أو تكبير أو تصغير العناصر.
مثال:
<style>
.box {
transition: transform 0.5s;
}
.box:hover {
transform: scale(1.2) rotate(10deg);
}
</style>

الدرس الثاني عشر: الرسوم المتحركة

الرسوم المتحركة في CSS تُستخدم لإنشاء حركات مخصصة على العناصر.

شرح مفصل:
  • @keyframes: لتحديد مراحل الحركة.
  • animation: لتطبيق الحركة على العنصر.
مثال:
<style>
@keyframes bounce {
0%, 100% { transform: translateY(0); }
50% { transform: translateY(-20px); }
}
.box {
animation: bounce 1s infinite;
}
</style>

الدرس الثالث عشر: الإعلامات والاستعلامات

الإعلامات تُستخدم لجعل التصميم مستجيبًا ويتكيف مع أحجام الشاشات المختلفة.

شرح مفصل:
  • @media: لتطبيق أنماط محددة بناءً على حجم الشاشة.
مثال:
<style>
body {
background-color: lightblue;
}
@media (max-width: 600px) {
body {
background-color: lightcoral;
}
}
</style>

الدرس الرابع عشر: تحسين الأداء

تحسين الأداء يُساعد في تسريع تحميل الصفحات وتحسين تجربة المستخدم.

شرح مفصل:
  • تجنب الأنماط المعقدة: استخدم أنماط بسيطة لتجنب التأثير على الأداء.
  • استخدام ملفات خارجية: قم بتخزين CSS في ملفات خارجية لتسريع التحميل.
  • تقليل CSS غير المستخدم: قم بإزالة الأنماط التي لا تحتاجها.

الدرس الأول: ما هو Bootstrap وكيفية إضافته؟

Bootstrap هو إطار عمل لتصميم واجهات المستخدم بشكل سريع وسهل باستخدام مكونات CSS وJavaScript جاهزة.

كيفية إضافة Bootstrap:
  • يمكنك إضافة Bootstrap عبر شبكة CDN باستخدام الرابط التالي داخل عنصر <head> في HTML:
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

الدرس الثاني: نظام الشبكة (Grid System) في Bootstrap

نظام الشبكة هو أساس Bootstrap، ويُستخدم لتقسيم الصفحة إلى أعمدة وصفوف لإنشاء تصميمات مستجيبة.

شرح مفصل:
  • يتكون النظام من 12 عمودًا.
  • يمكنك استخدام الفئات .row و .col لتحديد الأعمدة.
  • يدعم الأحجام المختلفة مثل .col-sm، .col-md، .col-lg.
مثال: <div class="container">
<div class="row">
<div class="col-sm-6">نصف العرض</div>
<div class="col-sm-6">نصف العرض</div>
</div>
</div>

الدرس الثالث: الألوان والخطوط في Bootstrap

Bootstrap يحتوي على مجموعة من الفئات لتنسيق النصوص والعناصر باستخدام ألوان وخطوط مدمجة.

الألوان الأساسية:
  • .text-primary: لون أزرق.
  • .text-danger: لون أحمر.
  • .bg-success: خلفية خضراء.
مثال: <p class="text-primary">هذا نص أزرق</p>
<div class="bg-success text-white">خلفية خضراء مع نص أبيض</div>

الدرس الرابع: المكونات (Components) الشائعة في Bootstrap

Bootstrap يوفر مكونات جاهزة مثل بطاقات، أزرار، قوائم منسدلة، وغيرها.

أمثلة:
  • البطاقات (Cards).
  • القوائم (Lists).
  • الأيقونات (Icons).
مثال: <div class="card">
<div class="card-body">محتوى البطاقة</div>
</div>

الدرس الخامس: الأزرار (Buttons)

الأزرار في Bootstrap توفر تصميمات جاهزة ومتناسقة.

أمثلة على الأزرار:
  • .btn-primary: زر أزرق.
  • .btn-danger: زر أحمر.
  • .btn-outline-success: زر بخط أخضر فقط.
مثال: <button class="btn btn-primary">زر أزرق</button>
<button class="btn btn-danger">زر أحمر</button>
