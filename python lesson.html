اساسيات لغه python

لغة Python هي واحدة من أشهر لغات البرمجة في العالم، وتُستخدم في العديد من المجالات مثل تطوير الويب، وتحليل البيانات، والذكاء الاصطناعي، وتعلم الآلة، وأتمتة العمليات، والكثير غيرها. تتميز Python بسهولة تعلمها وقراءتها، مما يجعلها اختيارًا مثاليًا للمبتدئين والمحترفين على حد سواء.

الدرس الرابع :الترابط في بايثون

الترابط (Threading) هو مفهوم يتيح تشغيل مهام متعددة (Threads) بشكل متزامن داخل عملية واحدة. في Python، يمكن استخدام مكتبة threading لإنشاء وإدارة الخيوط (Threads). يساعد الترابط في تحسين كفاءة البرامج من خلال تنفيذ المهام المتعددة في وقت واحد، خاصةً عند التعامل مع مهام إدخال/إخراج (I/O) مثل قراءة الملفات أو الشبكات.

شرح مفصل:
  • متى نستخدم الترابط؟عند الحاجة إلى تنفيذ عدة مهام في وقت واحد. لتجنب تعطيل البرنامج أثناء تنفيذ عمليات طويلة. لتحسين الأداء في التطبيقات ذات المهام المتزامنة.
    • مثال:
    import threading def print_numbers(): for i in range(5): print(f"Thread 1: {i}") # إنشاء وتشغيل الخيط thread = threading.Thread(target=print_numbers) thread.start() # تنفيذ التعليمات الرئيسية أثناء تشغيل الخيط for i in range(5): print(f"Main Thread: {i}")

الدرس التاسع:التنبؤ بالسلاسل الزمنية باستخدام Python

التنبؤ بالسلاسل الزمنية هو مجال في التحليل الإحصائي يستخدم لتحليل البيانات المتسلسلة زمنياً (مثل أسعار الأسهم أو درجات الحرارة) من أجل التنبؤ بالقيم المستقبلية. Python توفر العديد من الأدوات والمكتبات التي يمكن استخدامها للتنبؤ بالسلاسل الزمنية مثل Prophet، ARIMA، وLSTM.

شرح مفصل:
  • استخدام مكتبة Prophet للتنبؤ بالسلاسل الزمنيةProphet هي مكتبة مفتوحة المصدر تم تطويرها بواسطة Facebook، وهي سهلة الاستخدام وتسمح بالتنبؤ بالسلاسل الزمنية بشكل دقيق، حتى في حالة وجود بيانات مفقودة أو اتجاهات موسمية.
  • 2. استخدام ARIMA للتنبؤ بالسلاسل الزمنية ARIMA (AutoRegressive Integrated Moving Average) هو نموذج يستخدم للتنبؤ بالسلاسل الزمنية بناءً على القيم السابقة (التلقائية) والفروقات والتناوب.
مثال: # تثبيت المكتبة # pip install prophet import pandas as pd from prophet import Prophet import matplotlib.pyplot as plt # إنشاء بيانات السلسلة الزمنية (على سبيل المثال: بيانات شهرية) data = { "ds": ["2023-01-01", "2023-02-01", "2023-03-01", "2023-04-01", "2023-05-01", "2023-06-01"], "y": [10, 15, 13, 18, 22, 25] } # تحويل البيانات إلى DataFrame df = pd.DataFrame(data) df['ds'] = pd.to_datetime(df['ds']) # تأكد من أن الأعمدة بتنسيق تاريخ # إنشاء نموذج Prophet model = Prophet() # تدريب النموذج model.fit(df) # إنشاء بيانات مستقبلية للتنبؤ future = model.make_future_dataframe(df, periods=3, freq='M') # إضافة 3 أشهر للتنبؤ # التنبؤ بالقيم المستقبلية forecast = model.predict(future) # عرض النتائج model.plot(forecast) plt.show()
# تثبيت المكتبة # pip install statsmodels import pandas as pd import numpy as np import matplotlib.pyplot as plt from statsmodels.tsa.arima.model import ARIMA # إنشاء بيانات السلسلة الزمنية (على سبيل المثال: بيانات شهرية) data = [10, 15, 13, 18, 22, 25, 30, 35, 40, 45] # تحويل البيانات إلى DataFrame df = pd.DataFrame(data, columns=['y']) df['ds'] = pd.date_range(start='2023-01-01', periods=len(df), freq='M') # تحديد نموذج ARIMA model = ARIMA(df['y'], order=(1,1,1)) # p=1, d=1, q=1 # تدريب النموذج model_fit = model.fit() # التنبؤ بالقيم المستقبلية forecast = model_fit.forecast(steps=3) # التنبؤ بـ 3 فترات زمنية قادمة # عرض النتائج print("التنبؤات المستقبلية:", forecast) # رسم البيانات والتنبؤات plt.plot(df['ds'], df['y'], label='البيانات الأصلية') plt.plot(pd.date_range(start='2023-11-01', periods=3, freq='M'), forecast, label='التنبؤات', color='red') plt.legend() plt.show()
